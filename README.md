# Bootstrap 5.0 Starter Template

## Technology:

1. Bootstrap 5
1. BabelJS 7
1. PostCSS
    - Autoprefixer
    - PurgeCSS
1. Webpack 5
1. Webpack-dev-server

## First Install Dependencies

> ```sh
> npm install
> ```
>
> or
>
> ```sh
> yarn install
> ```

## Create Development Build

> ```sh
> npm run build
> ```
>
> or
>
> ```sh
> yarn run build
> ```

## Create Production Build

> ```sh
> npm run build:prod
> ```
>
> or
>
> ```sh
> yarn run build:prod
> ```

## Start Development Server at 9090 Port

> ```sh
> npm start
> ```
>
> or
>
> ```sh
> yarn start
> ```
